# Short Presentation Script

## Introduction

This project is a responsive Airbnb homepage clone built with React and Tailwind CSS. The goal was to convert a real-world travel marketplace interface into clean frontend code while keeping the layout responsive and user-friendly.

## Approach

I split the homepage into reusable React components: header, hero, category filter bar, listing grid, listing card, experiences section, host call-to-action, and footer. Listing content is stored separately in a data file, so the UI can be scaled or connected to an API in the future.

## Design

The design follows Airbnb-inspired patterns: rounded search controls, image-focused cards, clear pricing, ratings, and a clean white layout. I also added creative sections for experiences and hosting to make the project more complete than a simple visual copy.

## Responsiveness

The page is mobile-first. On smaller screens, the search form stacks vertically, categories scroll horizontally, and listings use a single-column layout. On larger screens, the layout expands into multi-column grids.

## Tools Used

The project uses React for components, Tailwind CSS for responsive styling, Vite for development and build tooling, and Lucide React for icons.

## Challenges

The main challenge was balancing clone accuracy with originality. I kept recognizable Airbnb-style patterns while adding unique content and sections suitable for an internship project.

## Learning Outcome

This project improved my understanding of component structure, responsive UI design, Tailwind utility classes, reusable data rendering, and frontend project documentation.
